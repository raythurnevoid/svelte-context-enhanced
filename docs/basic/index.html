Svelte Typed Context

Basic Usage

With createContext function you can create typed context setter and getter.

BasicExample.svelte 
<script lang="ts">
	import BasicExampleChild from "./BasicExampleChild.svelte";
	import { setContext } from "./BasicExampleContext";

	setContext("value");
</script>

<BasicExampleChild />
BasicExampleChild.svelte 
<script lang="ts">
	import { getContext } from "./BasicExampleContext";

	const value = getContext();
</script>

{value}
BasicExampleContext.ts 
import { createContext } from "@raythurnevoid/svelte-context-enhanced";

export const [setContext, getContext] = createContext<string>();

Result

value

API

createContext takes no arguments but wants you to set it's generic type in order to type the given context setter and getter. 

export declare function createContext<T>(): readonly [(context: T) => T, () => T];
